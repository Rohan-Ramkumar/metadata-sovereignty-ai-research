# AI Feature Evaluation Template
Metadata Sovereignty Alliance – AI Capability Review

This template helps evaluate AI tools and features for suitability within ethical, accessible, metadata-aware systems.

Students should complete this template when analysing potential AI technologies.

The goal is to assess:

• usefulness
• transparency
• accessibility
• governance risks
• metadata requirements

---
