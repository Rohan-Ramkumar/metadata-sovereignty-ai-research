# Contributing Guidelines

## Workflow

1. Create a branch for your work.
2. Add research or edits to the appropriate folder.
3. Include sources and citations.
4. Submit a pull request for review.

## Documentation Standards

Each research document should include:

Author:
Date:
Sources:
Summary:

## Citation

Use credible sources such as:

• academic publications
• government or NGO reports
• reputable technology documentation
• accessibility guidelines

Avoid unsourced claims.

## Version History

Major updates should include a short change note at the bottom of the document.
