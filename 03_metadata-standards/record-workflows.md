# Record Lifecycle Workflows

## Workflow 1 – Record Creation

1 Create document  
2 Record author and role  
3 Record creation date  
4 Record consent status  
5 Assign version number  

## Workflow 2 – Record Update

1 Open existing record  
2 Edit document  
3 Update change history  
4 Increase version number  

## Workflow 3 – Record Export

1 Export record  
2 Attach metadata  
3 Include authorship citation
